RHINO documentation
===================

RHINO is a translation layer, which allows switching between a coverage view and an object view within a data cube.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Features
--------

- Connects to the Open Data Cube
- Allows object-based (spatial) queries in addition
